# Fiji Worm Counter

## What This Project Does
Automates counting of GFP (green) and RFP (red) fluorescent worms from RGB TIFF images generated by WormScan. Replaces manual counting in Fiji (~30 min per class session).

## Image Format
- RGB TIFFs named `RGBImage_Plate_XXTIFF.tif`
- R channel = red (RFP) worms
- G channel = green (GFP) worms
- Yellow appearance in composite = R+G overlap
- Black background surrounds a circular plate region

## How to Run
```bash
pip install -r requirements.txt
# Place .tif files into scan/
python worm_counter.py
# Results in output.txt, annotated debug images in output/
```

### CLI Options
- `--min-size` — minimum contour area to count as a worm (default: 50)
- `--max-size` — maximum contour area to count as a worm (default: 5000)
- `--scan-dir` — input directory (default: `scan/`)
- `--output-dir` — output directory for debug images (default: `output/`)

## Dependencies
- Python 3.8+
- opencv-python
- numpy
